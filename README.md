# ThreatInsight AI

AI-driven phishing detection and cybersecurity platform.

## Tech Stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd threatinsight-ai

# Install dependencies
npm i

# Start the development server
npm run dev
```
